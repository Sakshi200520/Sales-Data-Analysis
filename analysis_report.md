# 📊 Sales Data Analysis Report
**Author:** Sakshi Sanap  

---

## 📌 Project Overview
This project analyzes sales data to extract meaningful insights such as total revenue, average sales, and best-selling products. It also includes data cleaning and visualization.

---

## 📂 Dataset Description
The dataset contains information about:
- Product names
- Total sales values

---

## 🛠️ Steps Performed

### 1. Data Loading
- Loaded dataset using pandas
- Handled file not found errors

### 2. Data Exploration
- Displayed first 5 rows
- Checked dataset structure
- Verified column names and shape

### 3. Data Cleaning
- Removed duplicate records
- Handled missing values by replacing them with 0
- Converted `Total_Sales` column to numeric format

---

## 📊 Analysis Results

- **💰 Total Revenue:** Calculated using sum of sales  
- **📊 Average Sales:** Mean of total sales  
- **📈 Highest Sale:** Maximum value in dataset  
- **📉 Lowest Sale:** Minimum value in dataset  
- **🏆 Best Selling Product:** Product with highest total sales  

---

## 📦 Sales by Product
Sales were grouped by product to understand performance of each item.

---

## 📉 Data Visualization
- Bar chart showing sales distribution by product
- Helps in identifying top-performing products visually
